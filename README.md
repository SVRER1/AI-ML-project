# BFS vs DFS Path Finding

## Overview

This project implements two fundamental search algorithms used in Artificial Intelligence: Breadth First Search (BFS) and Depth First Search (DFS). The purpose of the project is to understand how these algorithms explore a state space and how their approaches differ when applied to a grid-based path finding problem.

---

## Features

* Grid-based path finding
* Implementation of BFS for shortest path discovery
* Implementation of DFS for depth-based traversal
* Simple and readable code structure

---

## Concepts Used

* Uninformed search techniques
* State space representation
* Graph traversal methods
* Basic problem solving in AI

---

## BFS vs DFS Comparison

| BFS                      | DFS                           |
| ------------------------ | ----------------------------- |
| Uses a queue             | Uses a stack or recursion     |
| Explores level by level  | Explores depth first          |
| Guarantees shortest path | Does not guarantee optimality |
| Higher memory usage      | Lower memory usage            |
